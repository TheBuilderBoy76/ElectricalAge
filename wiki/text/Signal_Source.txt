}}Signal source is one of creative mode only blocks. It delivers electrical signal to devices.
==Usage and behavior==

Only signal cable can be connected to Signal source.

Signal source voltage can be set after right mouse click on it.
It can deliver signal voltage between 0V and 50V. You can set voltage above 50V but [[Signal Cable]] connected to it will explode.

The device have state indicator which lights up when voltage is equal or above 25V.
